JavaScript中的运算符

# 运算符

  1. 单位运算符 + - ++ -- null --> 0 undefined --> NAN
  2. 双位运算符 + - * / % += -= ...
  3. ? : 三目运算符 也被成为条件运算符
  4. () 调整运算顺序
  5. , 多个表达式 但是只有最后一个返回结果

运算符具有一定的优先级， 我们一定要根据优先级来进行计算 必要时可以使用() 来调整运算顺序。
